作为一名羽毛球新手选择啥样的俱乐部才合适？

发表于 2016-12-08 | 分类于 经验 |

本文编辑自 游游 投稿 1
作为一名嘎嘎新的羽毛球选手，选择一个好的俱乐部 ＝ 选择了一个积极健康的业余爱好社区氛围，可以尽量少走弯路、不受伤、快速提高，同时也能享受到羽毛球这项运动带来的快乐。

好的羽毛球俱乐部是酱婶儿的

有从新手入门练习到中高级对抗的梯次配置2 。类似的梯次配置在北京的忘不了、冰羽、网羽有约等很多俱乐部都有，有些俱乐部还会无偿提供新手培训3，也会为那些想在百尺竿头更进一步的会员提供付费教练和私教等有偿服务，比如北京的旭日星空羽毛球俱乐部就有成体系的教练团队，还有诸如升学加试、教练员培养等功能，为全国各大高校的体育专业和特招输送人才。

如何辨别黑心俱乐部

有一些号称自己经营了多少年的俱乐部，利用球友之间的矛盾获取私利，以为全天下都是傻子，主要特征：

  1. 管理尺度不一 场地有四片，拢共三十人。高手不花钱，也不和你玩。
  2. 群跟球友之间讨价还价 想跟高手打？在我这办卡。高手忙的话，人情欠得大。
  3. 双簧 长得真好看，衣服差一点。凭啥受刺激，最后你买单。
  4. 骗报班 想快速提高？体大有人教，一年八千块，交钱人不见。

新手、高手和俱乐部要有一个平衡。

  • 俱乐部承担人不够多费用不够成本的风险。在相对固定、对球友有信心、不谋取暴利的前提下，提供好球好场地。
  • 新手要足够勤奋、进取、谦虚，心中杂念少，坚持不懈并有提高。
  • 高手要对新手有信心并无私分享经验，督促新手进步。谁来为高手买单？以上三者其实共同承担促成高端俱乐部的风险、也成就了高端俱乐部的可能性。

另一个矛盾是所谓的高手其实也需要和同水平甚至更高水平的高高手一起锤炼提高，至少也得保持状态。
高水平俱乐部 ≠ 高端俱乐部。
负点责的高端俱乐部用好球并提供固定名额。比如使用金胜利 1 号、每块场地 ≤ 6 人来迎合需求，培养了很多忠实会员，堪称现代羽毛球俱乐部楷模。
一旦进入了这样的俱乐部，肃然起敬是不够的，还要付出努力和汗水，因为所有会员都共同承担着彼此在羽球生涯中的命运，互为载体。
些许汗水和球技提高之后，是各自会心的微笑。 只要良性发展，有理由相信现在的一些整体水平不高的俱乐部，以后也必然会发展成为高水平的高端俱乐部。
作为新手，推荐一些北京的高端俱乐部给大家，也欢迎那些负责的俱乐部管理者和群主把联系方式通过回复或消息发给我们。

  2. 说个误区，新手争先恐后的和高手打球并不能快速提高水平。在新手形成正确的动作定型前，过快的球速只能带来负面影响——动作变形、错误、疲于四处救球产生的运动伤害，轻则形成错误的动作习惯的天花板限制了水平提升，重则导致诸如肌肉韧带拉伤、关节不稳定和骨折等严重伤害。 ↩︎

回力和鸿星尔克鞋真是越做越好

回力和鸿星尔克的球鞋真是越做越好

第一次看回力还是初中，体育课上老师穿了双造型奇特的鞋——哎卧槽跳得老高了。那时就想有朝一日长大挣钱我也一定要买它一双！只是没想到十多年过去了一直被学业和工作所累。终于前段时间听朋友说日本有，就带双回力，顺便还整了鸿星尔克。 即便如此，也不得不佩服国外的先进技术。一穿上脚的瞬间就高潮了，有图为证： [gallery ids=”517,518” type=”columns” orderby=”rand”]

先说抓地

脚穿这双回力站在羽毛球场上，和小时候沙土地玩篮球打出溜滑儿的感觉完全不同。每当用核心力量将重心控制在两脚之间，身形突如其来的静止让人不适应；而鸿星尔克的抓地就要更自然一些，仿佛动静之间不存在什么转换，一切都是那么的自～然。

再讲缓冲

不得不说回力和鸿星尔克在缓冲上完全是两个极端，回力是一下陷下去——初上脚感觉保护非常到位，但是略软回弹性能略差，对于本来启动就慢的人是一种折磨；鸿星尔克的回弹就略给力，但穿起来感觉略硬。因此，对于年轻一些或软骨保护得好的人来讲，建议上鸿星尔克；如果已经过了生长期想要岁月静好细水长流的球友，个人十分推荐买回力！

小讲包裹

包裹就是跟脚儿，回力不管是记忆的鞋底还是后跟PHF技术非常不错，穿省力脱费力——简言之就是布的纹理比较强，缺点就是时间长了无论是袜子还是鞋后跟内侧都容易起球。但为了合脚，这点问题可以忽略；鸿星尔克这双SS的包裹感就不很强烈，但是也已经到了返璞归真的地步，与YONEX、VICTOR和李宁的鞋相比完全不落下风。

写在最后

如果你没我这么事儿逼，恰好又懂得正确使用步法、落地前先弯腿的话，任何羽毛球鞋——甚至穿普通运动鞋都可以在羽毛球场上自由的飞翔。况且YONEX、VICTOR和LI-NING也并非不思进取。

真相

后来有朋友说我的回力其实叫亚瑟士，我的鸿星尔克也不是鸿星尔克，而是叫做美津农。 请，别打死我！

科学上网1安装homebrew

  1. 打开“终端”粘贴脚本：/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)”
  2. 脚本会解释它的作用，然后在您的确认下执行安装。高级安装选项（英文）请看 这里2。

使用homebrew安装youtube-dl

  1. 在已经成功安装homebrew的终端里，输入brew install youtube-dl
  2. 安装好_youtube-dl_后即可使用_youtube-dl_命令下载视频（支持批量下载），比如你要下载最新的德国公开赛视频，则可以使用如下命令3：youtube-dl https://www.youtube.com/playlist?list=PLA7ZcagI0frD6kWiFlcZGBCCRq5vC\_2\_H
  3. 可以点赞了。

_ 　注意一下只要是youtube-dl这个脚本支持的平台，比如Linux/Unix甚至Windows下都可以使用，并不局限于苹果，只不过苹果的环境稍微简单一些不容易出现问题。_

不要用互相伤害的方式来打羽毛球

发表于 2016-10-27 | 分类于 经验 |

上周在中国民航干部管理学院羽毛球馆又一位球友不幸离我们而去。在惋惜之余，对仍“冒着生命危险”和让人“冒着生命危险”打球的球友和群主们给予一些合理的建议，很有必要。

不要跟专业球员比，人家是 5.0 的排量。
首先，大众体育和竞技体育的目的不同，以健身、娱乐和提高社会交往为主，强烈建议活动组织带领活动参与者以正确的热身、运动和冷却的活动流程，采取一个在受控范围的递进式运动强度，以素质和水平匹配的方式去分配对抗双方。
其次，参与者在活动前需要确认自己的身体健康水平和情绪状态。例如已经醉酒或连续熬夜工作后，再去参加高强度的对抗基本上可以看成是作死的行为。
正确的羽毛球活动参与方式应该是酱婶儿的：

  1. 以不伤害自己和不被别人伤害为大前提，根据自己的作息时间、身体健康程度、情绪状态和技术水平选择相应的活动人群、时间和场所。很多俱乐部有分级对抗场地、新手教学场地、陪练场地等等都非常好。
  2. 提前15～30分钟到场进行科学的热身，热身到身体热、关节润滑液分泌水平提升，同时也给神经系统一个信号——大爷要运动了，各部门都精神点！
  3. 有球热身，可以从网前小球到中距离的推、抽再到离线的高远球，从慢速到快速，让拍线和身体逐渐适应大力量和高强度的输出。不要一上来就前后拉吊对手，这时的目的应该是检查自己技术动作的稳定性，从而确定和调整状态。比如，发现高远球的动作变形，则可以果断进行纠正，保证后面对抗时不会因此失分。
  4. 开始对抗后，第一局球不要发力过猛，有条件尽量以放松的心态和身体去击球，你会发现比紧张状态下的发力更充分。在对抗的过程中选择正确合理的动作去击球，在不受伤的前提下积极比赛。宁可丢分，也不能以变形的动作击球，要承认这个球自己处理得不好。
  5. 激烈对抗后要先下场走动，尽量不要直接坐下休息。休息间隙要保证身体的热度，尤其冬天在温度不高的球馆，一定要穿好衣裤保暖。如果有必要，在下次对抗开始前再进行一些热身活动。
  6. 活动结束后的冷却和拉伸。运动必然随之产生多余热量，为了保证体温恒定身体会打开毛孔大量排汗，如果没有必要的冷却，后果可想而知。拉伸的作用也非常明显，肌肉在运动后仍保持紧张会压迫血管、持续消耗，也不利于提升速度灵敏等素质。
    体育锻炼、运动健身应该是积极阳光的，如果感觉到压抑和孤独，一定是哪里出了问题。

世界羽联官方教学视频一级 02 启动

发表于 2016-09-17 | 分类于 经验 |

定义

起动就是在判断对手出手击球的方向和落点后，双脚前脚掌快速蹬地向已推算好的击球点获得一个初速度的过程。

分类

起动共分三种：

  1. 左脚在前
  2. 右脚在前
  3. 左右脚平行

通俗解释

对新手来说，起动就是“小跳”，方便衔接前后步伐。

纠错

在国际比赛中经常看见女单选手有明显的“跳”，造成这样的原因主要有两方面：

  1. 高速摄像导致动作看起来“慢”，看了电视里比赛觉得可以和国际球员一较高下的朋友请看低视角视频或现场看一次再下结论。
  2. 力量较男单弱，需要更长的距离蓄力。
  3. 根据单打节奏需要，保持连续运动速度，节省体力。

明确了启动后的移动方向，启动就变得不再漫无目的，而是有指向性。在爆发力相当的水平下，有方向的启动要比垂直起落的“跳”快很多。
