做搭档好处理的球，到搭档需要的地方去

发表于 2016-08-31 | 分类于 进阶技术 |


论前场搭档撤出的时机，这就是所说的与人方便，与已方便。

前场搭档

  1. 封网时发现对手防守弱点，以过网急坠的出球迫使对手起过或回球质量下降，便于后场搭档连续进攻。
  2. 封网时帮助同伴防后场的斜线底角，便于搭档在另一底角底线处杀球后上网。
  3. 搭档连续进攻时撤出封网位置转向后场进攻位置，便于搭档跟进，形成轮转。

后场搭档

  1. 进攻后略向中间回动（小跳一次），帮助搭档接两腰推球，推球对于前场是身后球，而对后场搭档来说是身前球比前场搭档更好处理。同样，处理后方便跟进形成轮转。
  2. 为何要轮转，轮转可以保持二人在场上有比较均匀且相对比较高的初速度，相对于静止再启动要节约体能。同时，轮转可以给对手增加出球线路选择的难度，也比较符合一般配合习惯。
  3. 帮助前场搭档防中腰位置的斜线球，形成轮转或双边压网的态势。

不打和打不好是两个概念

发表于 2016-08-31 | 分类于 经验 |

  • 不打＝想打的时候随时可以打，对手不可以不留精力防守；
  • 打不好＝打了也是被反击，造成的威胁基本可以忽略。

杀斜线就是这个道理，总杀直线对手会习惯，偶尔的一拍斜线就是变化，防守的人就必须有被爆斜线的觉悟。然而防守只会杀直线的进攻方就容易多了。
\<!– more –>
勾对角也一样。一般有这种机会都可以扑球得分了，为什么还勾呢？因为对方知道你有机会就会特别留意，所以这时候勾对角的威胁更大。这就是意识的空档。
如果没有稳定的勾对角技术，就会少一种变化，进而缺少造成主动的机会。
对于发球而言，到内角 1 号位、外角 2 号位、底线内角 3 号位、底线外角 4 号位和近身 5 号位都有不一样的战术作用，虽然发的最多的是 1 和 5 号位，但其他三个位置也要偶尔为之，高质量的变化会让接发的精力分散，有利于第三排主动。
同样的应用到封网上，凝而不发并不代表没有大力下压的能力，但一味的全力下压会让下一拍的连贯变慢、也会增加下网的几率，要知道水平稍高的防守都可以打出过网向下走的回球，那时候你的拍子还在网带以下该如何是好？
重点是意识和站位。

有没有一年就成为绝世高手的办法

发表于 2016-08-30 | 分类于 经验 |


很负责的讲，就算三分钟就能理解所有的原则和原理，也一定要花个两三年来练到全身肌肉也会。
\<!– more –>
这还不算，只要稍微有点放纵效果马上减半。
正确的练球历程应该是这样的：
❶ 首先要保证动作全对，打不打到球无所谓！ 逐渐练成正确的启动、移动步伐、击球技术、回动技术，错误的动作会造成出球限制和伤病。不过注意一点：对的动作不等于一样的动作，个人身体条件千差万别，所以适合自己身体条件的动作才是正确的——师傅领进门，修行在个人，具体请自行体会。
❷ 身体素质的锻炼不能停！ 反应速度、速度耐力、爆发力和绝对力量都不能欠缺。反应速度是所有体育活动都必须的、速度耐力决定多拍能力、爆发力是移动和击球的基础、绝对力量是杜绝伤病的有效法宝。
❸ 套路。 观察对手的技术特点，利用阴险的落点和节奏控制对手，在能力相当的情况下占据主动。
❹ 配合。
① 做搭档轻松就能处理的球。搭档在前场一定要下压，可选球路有杀球、吊球、挡边路远网，为搭档创造封网的机会；搭档在后场尽量一定要放网，可选择性使用勾两边或者压中间，为搭档创造下压机会。一旦被动起球，确保到位——够高、够接近边线和底线。如果选择平抽，抽球后一定要就近选择防守位，与接杀类似。
② 到搭档需要你的地方去。对方偷搭档后场，第一时间去封网。搭档下压就可以形成前封后攻的态势，即便搭档起球也有充分的时间再退直线防守。搭档在一个角落杀球，要记得保护另一个角落，否则对方抽、挑对角立刻被动。
③ 轮转。杀不动了，吊直线跟进封网，换搭档杀。后场能力差，吊直线跟进封网，让搭档去打。
❺ 稳定准确。 首先要打在界内，其次才是球速快、靠线贴网、过网就坠。这一点最难做到。 因为对于所有需要肌肉记忆、熟能生巧的技术而言，结局可能美好，过程一定枯燥。

双打，所有的勤学苦练都是为了下压

在这个问题上不应该存在争议，请收起你的玻璃心。
Victor官方微信的文章 写的不错可以扩展阅读，在这么多羽毛球微信号当中，好的文章烧纸也很难找到，在学习的过程中辨别很重要。 通俗地翻译，就是每一拍都要争取主动。
主动的目的呢？没错，不管是杀球还是做球，都是为了再下一拍更主动的下压、压到对方没有能力再回球。

双打发接发和抽球练习：拼了老命也不给对方起球

发表于 2016-07-06 | 分类于 经验 |


发接发不起球是双打要义，主要涉及发球、接发、抽球和压球技术：

  1. 发球，基本以前场球为主，要求过网下坠、速度快、落点尽量接近接发球线；可以偶尔发后场高球和快球破坏对方节奏。发球时持拍手在前，左手掌握方向和出球弧度，右手在有一致性的前提下控制速度，拍面与地面接近垂直。发球后根据质量和战术意图选择跟进封网或后退防守。
  2. 接发，第二拍球。决定第三拍是进攻、过度、防守的关键。
  3. 压球，包括切球、封网、吊球、杀球和抽球等，条件球尽量贴网而过之后后向下走。
  4. 抽球，有些也和压球一起叫抽压（抽丫）。感觉分开说比较好，因为这两个有交集单不是包含与被包含关系。抽球是中场过渡技术，抽球时两人平行站位，通过逐渐压球或突然改变节奏、方向和落点获得主动进攻机会。比如抽直线改斜线、抽直线改挡网、抽直线改平高球等等。获得主动时跟进，搭档相应后撤形成进攻姿势。抽球练习时以多拍为主，可以和封网一起练习。

发力技巧

  • 持拍手远离身体前伸，接近伸直
  • 掌心预留发力空间
  • 正手食指勾、后三指抓紧拍柄
  • 反手拇指顶、后三指抓紧拍柄
  • 节奏是放松-抓紧-放松，只有击球时抓紧
  • 尽量减少引拍动作
  • 球接近球拍再发力

报班学羽毛球真的有用吗？以怎样的姿势学才有用？

发表于 2016-07-05 | 分类于 经验 |

当然有用！
很多人学球也有个一年半载了，一个月4-8次课，一次两个小时，可是为什么感觉还是进步不大或者没有进步？
请自问除了上课的时候练球，其它时间还有什么练习。
步伐、挥拍、专项力量和专项体能这些……
可以猜到一定会有人说，我们不是专业的练那么正式干吗？我们就是快乐羽毛球。
放屁！ 花钱学球是为了快乐？捧教练么？
一节课两三个小时，教练在课上除了要纠正每个人的错误动作，其它时间就是在教学员怎么练，知道了以后自己得练……这是羽毛球培训最重要的内容。
另外也郑重的向一些练得少打得多的朋友介绍一句他们一定知道的话：“上课俩小时，一场就破功”，辛辛苦苦花两个小时初步定型的动作。
即便要“打”球，也要找一些水平低一些的人来打，这样你才会有时间去抠一些基本的动作。
动作定了型，才能去和高手对抗卡节奏。
这里说的动作不仅包括在各种位置的一致性好的标准击球，也包括给搭档做球、合理的步伐选择和回动。
“师傅领进门，修行靠个人”，教练给你一些指导性的原则，让你能分辨动作的对错、节奏的轻重缓急、单双打的战术。
然而羽毛球技术怎么提高，多次重复磨炼肌肉的记忆力、多钻研交流才是王道。冥想确实对训练有提升，但只是辅助。
所有的项目和爱好道理都一样，技术不分专业和业余，只有业精于勤。到了正确认识自己的问题和怎么改正的阶段，教练的责任就只剩下支持和陪伴了。

技术｜封网 ≠ 挂在网上：最好的距离是留有余地

发表于 2016-07-04 | 分类于 经验 |


如果发生了这种接杀起球搭档要去补位
考虑到运动员能力，图中封网位置比较靠后。男双一般不会跨半场杀斜线，这个球本来要是杀宝哥的正手（宝哥的左手正手），而且是结合部，容易造成回球的不默契。
不过在有准备的老司机面前这些都是浮云，接杀时机和落点对蔡队而言相对舒服，而且杀球位置非常靠近底角，所以起球的选择是直接回另一个底角。这样杀球方选择横向移动并下压需要移动差不多一个球场（横向）的距离并且需要额外蹬转。
然而从封网位置来看这个球相对容易，选择突击会形成持续进攻。

正确的封网应该是酱婶儿的，注意看脚
所以，推荐搭档杀球时封网站位不要超过发球线太多，考虑如下：

  • 发生图一中的情形，搭档容易补位形成持续下压
  • 我方杀球质量不高或落点提前被判断时转为平抽相对容易
  • 人眼注意区域是扇形，太靠网前会降低有效视野
  • 对方失误回了半场球可以直接扑杀

很多人可能担心突然的网前球会接不到，这个担心是完全多余的。最后一句话送给大家：

做搭档好处理的球，到搭档需要的地方去。

羽毛球双打接杀心法口诀：老司机都这么干

发表于 2016-06-30 | 分类于 经验 |


接杀要考虑的因素很多但大的原则只有几个，接杀重点在腿不在手，身法更重要。那时候李龙大的搭档还不是柳延星，陈文宏也没搭古健杰。以下口诀基本可以看上图验证：

  1. 压低重心别站死，看点卡位别蒙圈
  2. 前场防吊后防杀，站位在后心在前
  3. 接杀尽量用反手，正反切换太麻烦
  4. 右脚在后有空间，平抽回压撇斜线
  5. 接杀伸手别太紧，反手拇指顶宽面
  6. 借力前推弹后场，发力根据球速算
  7. 力量不够用腿补，千万不要伤到肩

先看来球！先看来球！先看来球！重要的事三遍，下面稍微解释一下：

  • 第1条说的是先跑位然后在正确的点接杀，站死一点想要接杀所有球是不可能的。
  • 第2条的位置要综合考虑，如果你们的上一拍出球质量差，那对方杀球质量就会相对比较高尽量靠后接，以防起为主; 如果对方是强杀或者你判断出对方会习惯性重复杀一个点和高度，那就果断前压寻找变化甚至通过挡网反抽反守为攻。
  • 第3条也不是绝对，以顺为主。
  • 第4条主要用于接持拍手肩部落点的杀球，所谓的前是相对于对方杀球的出手点，而不是球网。
  • 第5条见仁见智，不考虑动作一致性和假动作，直接往前推就好。主要的目标是出球质量高、收拍快、发力突然、动作小。
  • 第6条就是经验了，要多接杀才会有感觉。
  • 第7条，很多人接杀用大跳，你还没落地对方就第二拍杀球到了。所以即使是发力，也要考虑回球的速度和节奏。

以上是在教练指导、观看大赛视频和日常打球中对双打接杀的理解，可能有很多技术并不适用于所有人，如果遇到此类情况请千万不要客气，直接回复就好。

突击步法中国跳和马来步都是些个什么鬼？其实就是卡时间的小套路

发表于 2016-06-29 | 分类于 经验 |

有些人就是喜欢发明新名词，就像标题里的“什么鬼”，还有醉了、请允悲之类，中国跳和马来步也一样。其实都没什么关系，只不过创造这些词的人看国家队那么跳，看大马那么迈步子。
\<!– more –>
一切的步法都是卡时间，为了能在适当的时候以适当的速度出现在适当的地点以适当的击球手法出球到适当的地点。
为何网前接快速吊球要并步？
为了严谨加了个“快速”，因为在现在的技术体系里慢速的吊球几乎没有威胁。因为并步可以两次加速一步到位，尤其是往反手的方向。
其它快速的回球也一样； 为何起高远球之后可以慢慢的后退准备接球，那是因为球上天了！你和对手都碰不到，过早回中会让你的速度下降太多不利于下一拍启动；
为何应对一些低位的推球和抽球要下蹲？那是因为没有另外一种更快速又能保持主动的回球方式；
为何突击有时连身体都没有时间蹬转，那是因为机会稍纵即逝。其实不蹬转也一样并步回动，有空可以看看视频，很多人突击都是这样。
合理利用步法的目的是为了更好的击球，好的步法只有顺和科学分配体力两个要点。
只要时间和节奏允许，你也可以编一些属于自己的步法。 每一个现象级的球员都有自己对场地、步法、节奏甚至装备的理解，理解越透彻技术高度就越高。
相信大家在林丹、李宗伟、拉查诺、风云、山万、钻石等著名球员和组合身上都能看到这些创作的痕迹。
每一次技术的进步都会带来成绩的飞跃，希望大家不要那么狭隘，好好玩。
