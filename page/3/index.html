与其说林丹打的是太极球不如说他用的是九浅一深

发表于 2017-02-18 | 分类于 经验 |

本文中说到的”九浅一深”是在对抗活动中使用频繁的一种技巧，对想体验赢球快感的搭档双方都会起到很重要的作用。

想歪的去面壁。

九浅一深的前提

训练和实践的区别

只有在重复练习法当中——例如多球训练时——才需要固定的速度、力度和频率来保证锻炼条件固定。而在对抗中寻求变化，打破思维定势，不落入训练时的固定套路当中，才能达到事半功倍的效果。

决断准确

参考对方的位置、主被动情况、双方相对比较有威胁的技术，迅速判断出球的速度、方向和落点。

如何训练？

反应训练

听、看信号后快速反应，有球和无球练习都可以。如果没有十足的把握，就在对方击球后再移动。

步法练习

迅速、正确的启动方向和合理的步法衔接，可以争取到更合理的击球位置。

位置的选择

击球时要移动到的最佳位置并不是球的落点，而是距离球落点合适的距离，在这个距离上出球的选择最多、同时相对最舒服。

出手果断

时机稍纵即逝，昨天点杀斜线+突击直线的战例证明——鸡屎球路再好，错过也是白搭。

九浅一深的套路

所谓的“浅”就是距离网带比较近的位置，一般落点位于发球线以前同时具备一定的下落速度。 所谓的“深”含义有两个，一个就是位置的“深”，另一个是谐音“身”——意思就是追身。

先吊再杀

双打对抗中有防守能力的人都不太忌惮同水平的杀球，而连续的杀球会造成“抗杀性”——对面的小伙伴适应杀球速度并伺机反击。这时候先调动对手上网，再利用位置和时机的优势解决战斗。

先吊球之后会发现很多机会，一般人打出质量差的球着先想到的是防守。

网前缠斗背后绝杀

积极抢搓、抢放，通过网前逐渐的积累优势造成对方起球或网前回球质量变差，再由后场搭档杀球或网前搭档推扑。

好的发接发是得分的一半。

先点杀后重炮

在进攻机会不是特别明显又不想丢掉进攻优势的时候可以寻找比较好的落点连续点杀，待对方回球失误时痛下杀手。

九浅一深的心法

在所有体育运动中，科学的体能分配都是胜利的关键。总的体能储备是一定的，能巧妙的用少量的体能去消耗对方较多的体能，才能在关键分上稳定发挥、压制对方。
这个打法，尤其适合不再青春年少的老将们使用。

抠细节：羽毛球的移动周期

发表于 2016-12-18 | 分类于 经验 |

事物在运动、变化过程中，某些特征多次重复出现，其第一次开始至结束的这段时间就称为“周期”。

羽毛球的移动周期，就是从对方击出一拍球之后到其下一次击球间的这段时间。 很多球友都会经历抠细节的阶段，常深陷其中不能自拔，偶尔也会遇到一些难题。比如高、吊、杀的动作一致性和发力差异，出球速度与收拍、回位快慢，主、被动及相应的取位等等。 体育运动的通则就是手眼身法步，理论需要系统学习，认知偏差要及时纠正，下面结合个人浅薄的实际应用简单总结羽毛球一个动作循环当中的各个环节，希望能帮助到一些球友，也欢迎回复指出不足帮助完善。

移动的四个环节：

  1. 准备阶段（判断速度和高度、测距、落点、定形势、取位）
  2. 移动阶段（架拍、启动、跑动、观察）
  3. 击球阶段（蹬转、引拍、手法、发力）
  4. 回动阶段（收拍、步伐连贯、拉重心、回中）

注意事项

  1. 低蹲快架早出手。快速移动时重心尽量降低便于在身体的相对高点击球，早些架拍处理追身球和杀球时更得心应手，高点不等人晚打容易错过。
  2. 提前移动被晃走。如果不能准确预判，乖乖等出手。
  3. 合理握拍是关键。在场地的不同位置处理不同高度和速度的球，握拍要适应调整。
  4. 杀球用腿不用手。胳膊拧不过大腿就是这个道理，如果能借到腿的力，嘿嘿。不过前面说的不用手，意思是重杀需要多从腿上借力，第一请不要抠字眼儿，第二这个是个人理解欢迎不同意见。

终于以 34 岁高龄成功在一次考试中及格

发表于 2016-12-13 |

一辈子总要有体育项目可以坚持到老，全民健身也是必然趋势。作为一个老少皆宜的项目，羽毛球不会像跑步一样受到阴晴雨雪雾霾等天气环境的制约，强度和时间又高度可控，作为一个终身运动再好不过。 个人感觉，想要从业的话，得有个证。第一能让别人大概了解能力水平，第二是给自己一个交待。在我这样的年纪，显然赢得一场高水平比赛的难度要比考取一个职业资格证要大得多。 我也算是个爱学习的人，所以理论的培训比较注意，生怕以后给人讲错。不过这次考试的难点就在于技能评定，对我来说发高远球和对打高远球的难度远高于发网前球和前后场的挑吊组合。也可能我就是贱，有简单的不练…… 一周不到的准备时间，五天的指导课。加上并不怎么充足的课下练习，以及高远球 20％ 的命中率能得到技评 61 分我已经非常满足了。除了 5 分钟临场指导以外，其它的科目发挥都不算令自己满意。 于是在接下来的时间里，还是要着重的加强一下自己的基础。学球时的一位教练跟我强调过基本技术稳定性的重要，当时我正在给他的学生放水也没太在意。而最近半年参加了几次比赛，让我重新想起了他的这番话。 这充其量只能算转行体育事业之后的一小步吧。接下来，希望我能在做教学视频、临场指导和平时训练的时候尽量少出问题，两年之后顺利晋升。
