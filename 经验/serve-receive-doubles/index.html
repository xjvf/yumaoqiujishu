双打发接发和抽球练习：拼了老命也不给对方起球

发表于 2016-07-06 | 分类于 经验 |


发接发不起球是双打要义，主要涉及发球、接发、抽球和压球技术：

  1. 发球，基本以前场球为主，要求过网下坠、速度快、落点尽量接近接发球线；可以偶尔发后场高球和快球破坏对方节奏。发球时持拍手在前，左手掌握方向和出球弧度，右手在有一致性的前提下控制速度，拍面与地面接近垂直。发球后根据质量和战术意图选择跟进封网或后退防守。
  2. 接发，第二拍球。决定第三拍是进攻、过度、防守的关键。
  3. 压球，包括切球、封网、吊球、杀球和抽球等，条件球尽量贴网而过之后后向下走。
  4. 抽球，有些也和压球一起叫抽压（抽丫）。感觉分开说比较好，因为这两个有交集单不是包含与被包含关系。抽球是中场过渡技术，抽球时两人平行站位，通过逐渐压球或突然改变节奏、方向和落点获得主动进攻机会。比如抽直线改斜线、抽直线改挡网、抽直线改平高球等等。获得主动时跟进，搭档相应后撤形成进攻姿势。抽球练习时以多拍为主，可以和封网一起练习。

发力技巧

  • 持拍手远离身体前伸，接近伸直
  • 掌心预留发力空间
  • 正手食指勾、后三指抓紧拍柄
  • 反手拇指顶、后三指抓紧拍柄
  • 节奏是放松-抓紧-放松，只有击球时抓紧
  • 尽量减少引拍动作
  • 球接近球拍再发力
